# riverpod_starter_project

A Template project for small to medium Flutter Apps using [riverpod](https://riverpod.dev/) state management.

## Getting Started

### Project structure
```
- lib
  - core
    - config
    - models
    - repositories
      - impl
      - interfaces
    - services
      - impl
      - interfaces
    - style
    - widgets
  - features
    - example_feature
      - providers
      - views
      - widgets
  - app.dart
  - injection container
  - main.dart
```
